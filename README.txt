ProjectM is a Desktop Chat Application.

Client :
Start it with : python3 /Client/client.py

Create a new Account and Connect to a Server or Add a new one, then start chatting.


Server Side :
Start it with : python3 /Server/projectM_Server.py

Default param : localhost:4242 and stays online up to 5min.
